# UAT Checklist

- [ ] Student registration and login work.
- [ ] Teacher registration creates `pending_approval`.
- [ ] Teacher cannot login before admin approval.
- [ ] Admin can approve/reject teacher.
- [ ] Admin can create subjects.
- [ ] Teacher can create notes and quizzes.
- [ ] Student can list notes and open PDF.
- [ ] Student can view quizzes and submit answers.
- [ ] Student can submit doubt with selected teacher.
- [ ] Student can submit doubt without selecting teacher and auto-assignment runs.
- [ ] Student and teacher can exchange doubt messages with polling.
- [ ] Teacher can resolve doubt.
- [ ] Student can rate teacher only after resolution.
